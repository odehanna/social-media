Social Media Scanners provide full spectrum social media monitoring for critical locations and issues. Scanners can be used for environmental monitoring, issue tracking, and discovery. They can also be used for more intensive, targeted investigative work. SM Scanners are built on carefully tailored social media collections, designed to client needs and specifications.

SM Scanners allow users to track social media activity by hashtag and by the type of device used to post content. The User Dossier function enables close review of user activity and key biographical information.
